# Personal Portfolio Website

## Project Overview
This is my personal portfolio website, designed to showcase my skills, experience, and online presence. It serves as an introduction to who I am and provides ways to contact me.  

## Features
- **Name and Photo:** Displays my name (Nuhamin) and profile photo.
- **About Me Section:** A short bio highlighting my coding journey, passion, and current internship at DevTech.
- **Skills Section:** Divided into Technical Skills and Soft Skills, with a vertical line separating them for clarity.
- **Contact Section:** Includes Email, Instagram, and LinkedIn links with a center-aligned stair-step format.
- **Interactivity:** A scroll-to-top button for better navigation.

## Goal
The goal of this project is to create a personal online profile that demonstrates my growing web development skills and allows viewers to learn more about me.

## What I Learned
- **HTML:** Structured the website with semantic elements (header, sections, footer).  
- **CSS:** Styled the website with colors, gradients, flexbox layout, responsive design, and hover effects.  
- **JavaScript (Optional):** Implemented a smooth scroll-to-top button for enhanced user experience.

## Technologies Used
- HTML  
- CSS  
- JavaScript (for animations/effects)
